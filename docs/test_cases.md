# Тестовые сценарии и тест-кейсы

## TC-001. Запуск приложения и пропуск онбординга

| Поле | Значение |
|---|---|
| ID | TC-001 |
| Название | Запуск приложения и пропуск онбординга |
| Предусловия | Приложение Wikipedia установлено на Android-устройство |
| Приоритет | Высокий |
| Тип | Smoke |

Шаги:

1. Запустить приложение Wikipedia.
2. Если отображается экран онбординга, нажать кнопку Skip / Пропустить.
3. Дождаться открытия главного экрана.

Ожидаемый результат: главный экран приложения открыт, поле поиска доступно.

Фактический результат: главный экран открывается, поиск доступен.

Статус: Passed.
